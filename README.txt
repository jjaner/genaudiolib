GENAUDIOLIB - Generic audio Processing Library 

Jordi Janer, MTG-UPF 2016

This is a Matlab library to perform generic audio processing functions based on the Short-time Fourier Transform. It provides a number of functions and tw o application examples. 
Application examples required 3rd party libraries (NMFLIB, MIDILIB and YIN).

This library is intended for research and teaching purposes.
